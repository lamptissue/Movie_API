Movie - API

A user will be able to receive information on movies, directors and genres. The user is also able to create a profile and save data about their favourite movies.

Request URL HTTP Method Query parameters Request body data format Response body data format
Return a list of ALL movies to the use /movies GET None None A JSON object holding data about all movies: {
"title": "The Shawshank Redemption",
"year": "1994",
"genre": {
"name": "Drama" },
"director": "Frank Darabont"
}
Return data about a single movie by title to the user /movies/[title] GET :title None A JSON object holding data about the requested movie: {
"title": "The Godfather",
"year": "1972",
"genre": {
"name": "Crime"},
"director": "Francis Ford Coppola"
}
Return data about a genre by name/title /movies/genre/[name] GET :genreName None A JSON object holding data about the requested genre: {
"name": "Drama"
}
Return data about a director by name /movies/directors/[name] GET :directorName None A JSON object holding data about the requested director: {
"Peter Jackson"
}
Allow new users to register /users POST None A JSON object holding user data: {
"name": "Jane",
"favouriteMovie": []
} 		A JSON objects holding new user data: {
"name": "Jane",
"favouriteMovie": [],
"id": "2913dc82-92eb-44b5-b97c-b0449995d0da"
}
Allow users to update their user info /users/[username] PUT :id A JSON object holding new user's name:
{
id: 1,
name: "Paul",
favouriteMovie: [],
} 		A JSON object holding new user's data: {
id: 1,
name: "Paul Updated",
favouriteMovie:[],
}
Allow users to add a movie to their list of favorites /users/[id]/[movieTitle] POST :id, :movieTitle None A JSON object holding updated data: {
id: 1,
name: "Paul Updated",
favouriteMovie:["The Godfather"],
}
Allow users to remove a movie from their list of favorites /users/:id/:movieTitle DELETE :id, :movieTitle None A JSON object holding the removed data: {
id: 1,
name: "Paul Updated",
favouriteMovie:[],
}
Allow existing users to deregister /users/[id] DELETE :id None Message confirming user has been deleted - User (id) has been deleted
